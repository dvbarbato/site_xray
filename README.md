# site_xray

# Objetivo

Aplicação desenvolvida com Streamlit para prever e classificar imagens de radiografias normais e de pacientes com pneumonia.

# Tecnologias

  - Python
  - Streamlit
  - Numpy
  - Tensorflow
  - Pillow
  - OpenCV
  - Joblib
  - Scikit-learn
